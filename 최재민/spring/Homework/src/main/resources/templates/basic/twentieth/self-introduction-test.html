자기 소개
변경에 유연하고 일관성있는 프로그램을 작성하기 위해 애자일 프로세스 기반으로
프로그램을 작성하는 방법을 중접으로 학습 중에있으며 코드를 작성할때 SRP 규칙을 지키면서 작성하기 위해 노력 하고있습니다.
또한 가장 대중적으로 사용중이며 재사용과 확장이 가능한 개발 스킬을 향상 시킬수있고
Java 기반에 웹 어플리케이션을 개발하기에 표준과도 같은 Spring Framework 를 학습중입니다.
나아가 단일 컴포넌트 기반으로 러닝 커브가 비교적 쉬운 Vue 를 활용해 기존서비스에 점진적인 개선을 할수있도록 학습중에 있습니다.
때문에 저는 가까운 미래에 보다 나은 성능과 서비스의 안전성이 필요한 복잡한
기업용 엔터프라이즈 시스템을 제대로 구축하고 사용할수있는 개발자가 될것입니다.
